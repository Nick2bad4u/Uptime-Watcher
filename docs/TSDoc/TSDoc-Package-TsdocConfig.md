<h1>
@microsoft/tsdoc-config
</h1>
</header>
<p>
`<a href="https://www.npmjs.com/package/@microsoft/tsdoc-config" target="_blank" rel="noopener noreferrer">`{=html}NPM
package`</a>`{=html}`<br>`{=html}
`<a href="https://github.com/microsoft/tsdoc/blob/main/tsdoc-config/CHANGELOG.md" target="_blank" rel="noopener noreferrer">`{=html}CHANGELOG.md`</a>`{=html}
</p>
<p>
The `<strong>`{=html}@microsoft/tsdoc-config`</strong>`{=html} package
is an optional add-on for loading the
`<strong>`{=html}tsdoc.json`</strong>`{=html} file format that enables
users to define custom TSDoc tags. (This functionality was moved to its
own package because it requires external dependencies such as NodeJS and
`<code>`{=html}ajv`</code>`{=html}, whereas the main package is fully
self-contained.)
</p>
<h2 class="anchor anchorWithStickyNavbar_Dpyi" id="creating-config-files">
Creating config
files`<a href="#creating-config-files" class="hash-link" aria-label="Direct link to Creating config files" title="Direct link to Creating config files">`{=html}​`</a>`{=html}
</h2>
<p>
The `<strong>`{=html}tsdoc.json`</strong>`{=html} file is optional. When
used, it is expected to be found in the same folder as the
`<strong>`{=html}tsconfig.json`</strong>`{=html} file for a project. The
loader looks for it by walking upwards in the directory tree until it
finds a folder containing
`<strong>`{=html}tsconfig.json`</strong>`{=html} or
`<strong>`{=html}package.json`</strong>`{=html}, and then it attempts to
load `<strong>`{=html}tsdoc.json`</strong>`{=html} from that location.
</p>
<p>
The `<strong>`{=html}tsdoc.json`</strong>`{=html} file conforms to the
`<a href="https://developer.microsoft.com/json-schemas/tsdoc/v0/tsdoc.schema.json" target="_blank" rel="noopener noreferrer">`{=html}tsdoc.schema.json`</a>`{=html}
JSON schema. It defines tags using similar fields as the
`<a href="https://github.com/microsoft/tsdoc/blob/main/tsdoc/src/configuration/TSDocTagDefinition.ts" target="_blank" rel="noopener noreferrer">`{=html}TSDocTagDefinition`</a>`{=html}
API used by `<code>`{=html}TSDocParser`</code>`{=html} from
`<strong>`{=html}@microsoft/tsdoc`</strong>`{=html}.
</p>

</p>
